Face Mask and Social Distancing Detection
using VGG19

Developed a Deep Learning Model to detect if a person is wearing mask or not and check if social distancing norms were being followed. • Object Detection was carried out using Haar Cascade Model trained to detect faces to obtain the bounding box coordinates of faces in an image.
